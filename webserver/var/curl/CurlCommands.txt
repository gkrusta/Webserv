- to upload a photo:
curl -X POST -H "Content-Type: image/jpg" -F "file=@/Users/gkrusta/Desktop/d.jpg" localhost:8080/www/

- to upload text file:
curl -X POST -H "Content-Type: text/plain" -d @data.txt http://localhost:8080/www/

- to delete a directory (Forbidden):
curl -X DELETE http://localhost:8080/delete_this

- to delete a file:
curl -X DELETE http://localhost:8080/delete_this.jpg

- to reach a website:
curl -X GET http://localhost:8080/default/index.html

- to test error page:
curl -X GET http://localhost:8080/default/notexisting.html

- to test default error page:
remove error_page from config and default page will be shown
